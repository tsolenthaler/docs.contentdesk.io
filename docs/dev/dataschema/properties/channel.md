# channel

Spezifisches Properties für das definieren für welchen Ausgabekanal das Objekt verwendet werden soll.

## Beispiel / Use-Case

### Gästekarten / App

#### Akeneo API

``` json
{
    ...,
    "values":{
        "channel": [
            {
                "locale": null,
                "scope": null,
                "data": [
                    "app",
                    "ecommerce"
                ]
            }
        ],
    }
}
```
